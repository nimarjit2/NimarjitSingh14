📊 Financial Data Visualization using Python

As part of my data analytics coursework, I visualized the financial statement data of Ashok Leyland using Python.

The financial dataset was imported from a CSV file and analyzed in PyCharm. Using Seaborn and Matplotlib, I created bar charts to represent financial variables across multiple years.

📈 Financial variables visualized:

Equity Capital

Reserves

Borrowings

Total Assets

🔎 Steps performed:

Imported CSV data using pandas

Reshaped the dataset using melt()

Created a bar plot using seaborn.barplot()

Displayed the graph using matplotlib

💡 Insight:
Data visualization helps identify financial growth patterns and trends in company balance sheet data over time.
